# Изображения, видео и ссылки

## Добавление ресурсов

Если возникает необходимость добавить ресурсы (изображения, видео) в документ, сначала нужно создать папку в директории с секцией, в которой находится документ.

```
xcard-docs/
├── docs/
│   ├── new-section/
│   │   ├── img/ # папка с изображениями (название папки может быть любым)
│   │   ├── _category_.json
│   │   ├── <название_документа>.md
│   │   └── ...
│   └── ...
```

:::tip

Для изображений лучше назвать папку `img`, для видео - `video`.

:::

### Изображения

После создания папки, можно добавлять изображения в папку `img`.

```
xcard-docs/
├── docs/
│   ├── new-section/
│   │   ├── img/
│   │   │   ├── logo.svg # изображение
│   │   │   └── ...
│   │   └── ...
```

Для добавления изображения в документ, необходимо использовать следующий формат:

```md
![Описание изображения](./img/logo.svg)
```

На странице будет отображаться следующее:
> ![Описание изображения](./img/logo.svg)

### Видео

:::important

Для добавления видео, документ должен быть в формате `.mdx`, а не `.md`. Также необходимо добавить в начало документа следующий код:

```mdx
import ReactPlayer from 'react-player';
```

:::

После создания папки, можно добавлять видео в папку `video`.

```
xcard-docs/
├── docs/
│   ├── new-section/
│   │   ├── video/
│   │   │   ├── video.mp4 # видео
│   │   │   └── ...
│   │   └── ...
│   └── ...
```

Для добавления видео в документ, необходимо использовать следующий формат:

```md
<ReactPlayer playing controls url="./video/video.mp4" />
```

Альтернативно можно добавить ссылку на видео:

```md
<ReactPlayer playing controls url="https://www.youtube.com/watch?v=dQw4w9WgXcQ" />
```

На странице будет отображаться следующее:
> <ReactPlayer playing controls url="https://www.youtube.com/watch?v=dQw4w9WgXcQ" />

### Ссылки

Для добавления ссылки в документ, необходимо использовать следующий формат:

```md
[Описание ссылки](https://www.google.com)
```

На странице будет отображаться следующее:
> [Описание ссылки](https://www.google.com)
